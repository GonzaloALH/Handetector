# KuwoTreko (ManoTendon)
Código base para el reconocimiento de mano mediante Mediapipe en Raspberry pi 5, para replicar el movimiento de apertura o cierre de mano derecha con el fin de replicar en la otra mano, mediante el uso de servomotores cuyos ejes van acoplados a monofilamentos de hilo integrados de la muñeca a la falange distal de cada dedo en un exoesqueleto de mano para suplir el movimiento de motor en pacientes afectados por ataque cerebro vascular (ACV).
