# KuwoTreko (ManoTendon)
Código base para el reconocimiento de mano mediante Mediapipe, para replicar el movimiento de apertura o cierre de mano derecha para replicar en la otra mano mediante el uso de servomotores cuyos ejes van acoplados a monofilamentos de hilo integrados en un exoesqueleto de mano para suplir el movimiento de motor en pacientes afectados por ataque cerebro vasculare (ACV).
